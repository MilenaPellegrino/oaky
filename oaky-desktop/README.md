# Oaky Desktop - Sistema de Gestión de Precios y Stock

Aplicación de escritorio desarrollada en Python para la gestión completa de productos, precios y stock de Oaky - Tienda de Ropa.

## Instalación

### Requisitos Previos
- Python 3.8 o superior
- pip (gestor de paquetes de Python)

### Pasos de Instalación

1. **Navegar a la carpeta del proyecto:**
```bash
cd oaky-desktop
```

2. **Crear un entorno virtual (recomendado):**
```bash
python -m venv venv
```

3. **Activar el entorno virtual:**
```bash
# En Linux/Mac
source venv/bin/activate

# En Windows
venv\Scripts\activate
```

4. **Instalar dependencias:**
```bash
pip install -r requirements.txt
```

5. **Ejecutar la aplicación:**
```bash
python main.py
```

## Tecnologías Utilizadas

- **Python 3.x**: Lenguaje de programación
- **PyQt6**: Framework para interfaz gráfica
- **SQLite**: Base de datos local
- **CSV**: Procesamiento de archivos CSV
